# 組み合わせ案：ヨーロッパ・親友旅（_example・公開見本）

> Phase 2（trip-combine）の出力イメージ。wishlist＋プロファイル＋基本情報から複数案を生成。実ファイルは `combinations.md`（非公開）。

## 前提
- 5泊6日 / ローマ3泊・フィレンツェ2泊 / 親友2名 / 予算35〜40万円/人 / 鉄道移動OK・事前予約派
- 🚩 外せないもの（マスト）：**コロッセオ／ウフィツィ美術館** → できる限り全案に確保（入らない案は⚠️で明示）

## 案A〈王道2都市・芸術満喫〉
- 日割り：Day1 到着＆夜の街歩き／Day2 古代ローマ／Day3 バチカン／Day4 移動＆ウフィツィ／Day5 ドゥオモ＆夕景／Day6 帰国
- 入った：コロッセオ・バチカン・ウフィツィ・ドゥオモ・トレヴィ／落ちた：ヴェネツィア
- 予算：約37万円 ／ 向き：定番をしっかり、緩急もほしい人（花子向き）

## 案B〈ローマ単都市でゆったり〉
- 日割り：ローマに5連泊。日帰りでティボリも
- 入った：コロッセオ・ローマ主要＋郊外1日／落ちた：フィレンツェ
- **⚠️ マスト未充足：ウフィツィ美術館（フィレンツェ）は単都市案では行けません。** 行くなら案A/Cのように1泊フィレンツェを足す必要あり
- 予算：約34万円 ／ 向き：移動を最小化してのんびりしたい人（ウフィツィを諦められる場合）

## 案C〈美術・建築テーマ深掘り〉
- 日割り：美術館・教会・建築を主軸に、ボルゲーゼ美術館やアカデミア（ダヴィデ像）も追加
- 入った：ボルゲーゼ・アカデミア／落ちた：トレヴィ等の定番一部
- 予算：約38万円 ／ 向き：アートにとことん浸りたい人

## 案D〈食い倒れ＋街歩き〉
- 日割り：市場・食堂・ジェラート巡りを軸に名所は厳選
- 入った：テスタッチョ市場・トラステヴェレ食べ歩き／落ちた：美術館の一部
- 予算：約36万円 ／ 向き：観光より「食と暮らし」を楽しみたい人

## 💡 プロファイルからの追加提案（根拠つき）
- **ボルゲーゼ美術館（要事前予約）**：花子は美術館志向と学習済み → ベルニーニ彫刻が見もの。Day3午後の候補に
- **ミケランジェロ広場の夕景**：「早起き苦手・夕景向き」の傾向 → フィレンツェ最終夜に最適
- **トラステヴェレ地区の夜カフェ**：「親友と夜にゆっくり話す時間」を好む → 1日組み込み推奨
